\subsection{Aufbau von TCP-Verbindungen und Durchführung von Anfragen}

Für alle Anfragen wurde ein einfaches Python Script verwendet, welches die TCP-Verbindung aufbaut, die HTTP Anfrage sendet und die Antwort empfängt \ref{code:connection}.

\codeblock{../connection.py}{Python Script zum Aufbau von TCP-Verbindungen und Durchführung von HTTP Anfragen}{code:connection}

Das Script nutzt für die Anfragen die  \texttt{httpx} Bibliothek \cite{httpx_http2}, welche die TCP Verbindung für HTTP Anfragen vereinfacht.
Es werden die HTTP Versionen 1.1 und 2 unterstützt, welche über die Kommandozeile ausgewählt werden können \ref{code:connection-help}.
Das Script gibt die Antwort des Servers auf der Kommandozeile aus.

\codeblock{../connection_help.txt}{Ausgabe der Python script Optionen über den Befehl \texttt{python connection.py --help}}{code:connection-help}
